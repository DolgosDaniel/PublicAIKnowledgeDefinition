# Example: complex-project — Nimbus Commerce

The multi-service instance of the PAIK standard: 3 services (frontend, orders-api, catalog-api),
5 teams, 4 environments across 2 regions, per-service configuration. Start at
[`paik/project.md`](paik/project.md).

See [`../simple-project/`](../simple-project/) for the minimal single-service version of the same
standard — the same doc types, just one file instead of one-per-service/one-per-region.
